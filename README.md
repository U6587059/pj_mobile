## Getting Started
วิธี ใช้งาน
1. โหลดไฟล์จาก github 
2. เเตกไฟล์ที่โหลดมา
3. ทำการเปิดไฟล์ใน Visual Studio Code
4. run "flutter pub get" ใน terminal
5. เปิด emulator ในเครื่องของคุณ (android sdk)
5. run "flutter run " ใน terminal
6. เริ่มใช้งานได้เลย
